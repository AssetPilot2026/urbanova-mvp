# 🚀 Project Overview

**Urbanova**'s goal is to make real estate investing more accessible, transparent, and intelligent through Web3. 

<img src="./public/urbanova.png" alt="Assetpilot" style="width:100%; height:auto;" />

---

## Project Overview

Urbanova’s MVP focuses on delivering the **core investment experience**:

1. **Property Discovery:** Users can browse tokenized properties.
2. **Investment via Crypto:** Users can invest in fractional shares using crypto wallets.
3. **On-Chain Ownership:** Fractional ownership is represented on-chain using smart contracts.
4. **Real-Time Performance Tracking:** Investors can view property metrics and returns.
5. **Wallet Integration:** Supports wallet-based authentication and transactions.

The MVP is **designed for early user feedback** and testing before full platform development.

---

## Features

- Fractional property ownership using **Ethereum-compatible smart contracts**
- Crypto payment gateway integration
- Real-time dashboard for **property performance**
- Admin interface for property management
- Automated distribution of rental income to token holders

## Installation

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Steps

```bash
# Clone the repository
git clone https://github.com/AssetPilot2026/urbanova-mvp.git

# Navigate to the project folder
cd urbanova-mvp

# Installation dependencies
npm install

# Start
npm start

```
